## 👨‍💻 About This Repo

This repository documents my personal TypeScript learning journey — covering core concepts with clean, well-commented code. Each topic is organized as a separate file for easy reference and revision.

---

## 📚 Topics Covered

### 🔹 Fundamentals

- **Primitive Data Types** — `string`, `number`, `boolean`, `null`, `undefined`
- **Arrays** — Typed arrays, string arrays, mixed arrays with union types
- **Tuples** — Fixed-length arrays with defined types per position
- **Objects** — Object types, optional properties (`?`), readonly fields

---

### 🔹 Functions

- **Normal Functions** — Parameter types & return type annotations
- **Arrow Functions** — Typed shorthand functions
- **Object Methods** — Functions defined inside objects
- **Callback Functions** — Typed callbacks with array methods like `map`

---

### 🔹 Operators & Syntax

- **Rest Operator (`...`)** — Gather multiple values into one parameter
- **Spread Operator (`...`)** — Expand arrays or objects
- **Destructuring** — Object & array destructuring with TypeScript types
- **Ternary Operator (`?:`)** — Inline conditional decision making
- **Nullish Coalescing (`??`)** — Fallback values for `null` / `undefined`
- **Optional Chaining (`?.`)** — Safe property access on nested objects

---

### 🔹 Type System

- **Union Types** — Accept multiple types (e.g. `string | number`)
- **Intersection Types** — Combine multiple types
- **Type Aliases** — Reusable custom type definitions
- **Interfaces** — Structured contracts for objects and classes
- **Generics** — Reusable, type-safe logic with flexible type parameters
- **Type Assertion** — Manually telling TypeScript what type something is
- **Type Narrowing** — Checking types at runtime before operations
- **Nullable Types** — Safely handling `null` and `undefined` values

---

### 🔹 Advanced Types

- **`unknown` Type** — Safer alternative to `any`, requires type checking before use
- **`void` Type** — For functions that return nothing
- **`never` Type** — For functions that never return (throws / infinite loops)

---
